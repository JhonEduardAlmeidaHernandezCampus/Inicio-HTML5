Movil First

esta metodología trata de dar prioridad al diseño de una web en versión móvil y, posteriormente, se adapta a PC.

De este modo, se comienza con la maquetación y estructuración de una web acorde a las medidas y parámetros
que presenta la pantalla de un dispositivo móvil.

Después se escalan los elementos que conforman la página web y se van transformando y ajustando
a la características de la pantalla de un ordenador.

¿Qué diferencias existen entre Mobile First y Responsive?

web Responsive
consiste en pensar y crear una web en formato PC
y finalmente, adaptar todos los componentes a versión móvil. Mobile First
trata de desarrollar el sitio web inicialmente en formato móvil
y la adaptación se experimentará en el PC.

¿Por qué es importante el Mobile First?

° Una de las ventajas que ofrece este método radica en la posibilidad de disminuir el tiempo de carga de la web.
Al incorporar elementos optimizados para dispositivos móviles la web presentará una velocidad de carga superior.
Esto repercutirá en la disminución de la tasa de rebote.

° Si la web se muestra con rapidez, el usuario mostrará una mayor intención de continuar con la navegación.

° Mejorará el rendimiento y la usabilidad de la web.
Al diseñar pensando en móvil se ofrecerá justo el contenido de valor que espera
encontrar el usuario y se eliminará la información superflua o innecesaria. ° Mobile First beneficiará positivamente en el posicionamiento SEO.
Es decir, la adaptación de una web en formato móvil es uno de los factores que Google
tiene en cuenta para otorgar una posición a la web en la página de resultados de búsqueda.

¿Cómo saber si tu web tiene una buena adaptación móvil según Google?

Google ofrece algunas alternativas para conocer si una web muestra una adecuada adaptación móvil.

° La pestaña de «usabilidad móvil» de Google Search Console podemos conocer si las páginas de una web están adaptadas a estos dispositivos.
° Igualmente, podemos utilizar la herramienta «prueba de optimización para móviles que presenta Google«.
