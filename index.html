Etiquetas de listas

Lista ordenada

Usa la etiqueta ol para construir lista lista ordenada.
Para señalizar al navegador los elementos de una lista tienes que usar la etiqueta li dentro de ol.

Objetivos

  1. Encontrar un trabajo
  2. Ganar mucho dinero
  3. Cambiarse de ciudad

Lista no ordenadas

Para construir una lista no ordenada usamos la etiqueta ul.
Las listas no ordenadas son de diferentes tipos: Casillas, puntos, y círculos.
El valor estándar en la mayoría de los navegadores son los puntos o bullets.

  • leche
  • queso
  • huevos
  • azúcar

Lista de definición

También puedes hacer listas de definición utilizando la etiqueta dl.
Esta lista dará la definición a la palabra especificada.
Para que la palabra sea mas evidente y que se distinga bien de la descripción se suele poner también en negrita.

Cheese
Palabra en inglés para queso
Computer
Palabra en inglés para ordenador
